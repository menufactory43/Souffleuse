# Patterns Remotion du repo (conventions à respecter)
- `interpolate(...)` toujours avec `clamp`. Easings = `settle`/`settleSlow`.
- Aléatoire = `random('seed')` de Remotion, JAMAIS `Math.random` (sauf le `Math.sin` cosmétique déjà présent dans Caret/Grain — toléré).
- Pas de `Date.now`. Frappe = tableau de frames pré-calculé par `random(seed)` (voir TIMES dans ActeSouffle/ActeMidLine).
- Frappe→ghost→Tab : voir ActeSouffle (mots ghost en `settleSlow`, blur→net, ghost→ink via interpolateColors, press via interpolate sur PRESS).
- Geste milieu de phrase : voir ActeMidLine (caret qui remonte, left/right split, ghost inséré au caret).
